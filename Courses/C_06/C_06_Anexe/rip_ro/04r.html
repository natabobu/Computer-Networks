Aplicatie Java pentru simularea RIP routing protocol bazata pe algoritmul Bellman-Ford.
Specificatii pentru aplicatie:
-------------------------------------------------------------

Retea Virtuala

  1. Noduri
  2. Pachete Floating
  3. Link-uri

Comenzi de Actiune

  1. Step Button : Ruleaza aplicatia pas cu pas.
  2. Reset Button: Initializeaza aplicatia.
  3. Clock : Arata fiecare pas cu ceas. (Ceasul se schimba la 30 secunde.)

Tabel

  1. Tabel pentru Fiecare Nod
  2. Destination : Destinatia de la nodul sursa.
  3. Cost : Costul toatal pentru fiecare legatura. El este setat initial cu 1.
  4. NextHop : Directorul vecinul Hop conectat .
  5. "-": "-" inseamna "Costul este infinit" sau "NextHop este necunoscut".

Schimbarea Starii

  1. Link Down: Click pe legatura,si ea devine failed-link-state. (sterge legatura)
