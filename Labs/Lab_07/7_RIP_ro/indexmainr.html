Sumar

Acest website este o parte din proiectele SC 546 Fall 2002, care descrie procesul de modelare,proiectul de aplicatie si rezultatele simularii "the Java applet of the RIP routing protocol" bazat pe algoritmul Bellman-Ford.

Despre Proiect

Curs: SC546 Computer Communications and Networks, Fall 2002
Profesor: David Starobinski
Titlul Proiectului: "The Java applet of the RIP routing protocol"
Membrii proiectului: Marcos Andres Diaz
Yasuko Iwai

Puncte principale

  • Studierea celui mai popular protocol de routare Least-Cost, RIP (Routing Information Protocol) si algoritmul sau.
  • Proiectarea si implementarea programului de simulare - Java applet of the RIP routing protocol- bazat pe algoritmul Bellman-Ford.
  • Presupune calitatile si defectele acestui protocol executandul.
